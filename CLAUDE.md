## Rules

- **Static site only.** No build tools, no npm, no frameworks. Pure HTML/CSS/JS.
- **Never edit binary data files** inside `cellbrowser/CEPsh_DV/` or `cellbrowser/Ventral/`. These were generated by the bioinfo core and are not to be modified.
- **Editable files:** `index.html`, `cellbrowser/index.html`, `cellbrowser/dataset.json`, `cellbrowser/CEPsh_DV/desc.json`, `cellbrowser/Ventral/desc.json`, `igv/index.html`.
- **To add IGV data:** populate the `igv/` directory and update `igv/index.html`. Update `index.html` to remove the "Coming Soon" badge.
- **To add a third dataset:** copy data into `cellbrowser/<name>/`, add an entry to `cellbrowser/dataset.json`, update the landing page.
- All changes must be committed before pushing. Follow root CLAUDE.md red lines for `git push`.
